# 👶 Baby Care Scheduler

A free tool that helps newborn families coordinate caregiving — give every family member time to rest with fair, science-based caregiving rotations. ❤️

---

## ✨ Features

- 🆓 **Completely Free** — No sign-up or login required
- 🔒 **Privacy First** — All data stored locally in your browser
- 📱 **Cross-Platform** — Works on desktop, phone, and tablet
- ⚡ **Smart Scheduling** — Auto-generates balanced caregiving shifts
- 🎨 **Simple & Friendly** — Warm, intuitive interface designed for tired parents

---

## 🚀 Quick Start

1. Open the website in your browser
2. Add family members (Mom, Dad, Grandma, Grandpa, etc.)
3. Set each person's available time slots
4. Click **"Auto-Generate Schedule"**
5. Manually adjust if needed, then export and share

---

## 📋 Features Overview

### Member Management
- Add 2–6 family members
- Assign roles (Mom / Dad / Grandma / Grandpa / etc.)
- Distinguish between **working** and **non-working** members

### Smart Default Rules
| Member Type | Weekdays | Weekends |
|---|---|---|
| Working | Evenings only | All day available |
| Non-working | All day available | Not available |

> All defaults can be customized for any time slot.

### Auto Scheduling
- Intelligently assigns care shifts
- Prioritizes each person's preferred time slots
- Automatically balances workload across members
- Supports locking specific time slots

### Export & Share
- Export as CSV spreadsheet
- Copy as text to share in family group chats
- View summary statistics at any time

## 💡 Usage Tips

### Recommended Browsers

| Platform | Recommended Browser |
|---|---|
| iPhone / iPad | Safari |
| Android | Chrome |
| Desktop | Chrome or Safari |

> ⚠️ **Important:** Schedule data is saved in your current browser's local storage. Stick to the same browser you started with. Avoid opening the app inside WeChat (data gets isolated in WeChat's built-in browser).

### Multi-Device Use
- Designate one device as the "schedule manager" device, **or**
- Use the export feature to share the schedule with family members
- Automatic cross-device sync is not currently supported

### Supported Browsers
- ✅ Chrome (Windows / Mac / Android)
- ✅ Safari (Mac / iPhone / iPad)
- ✅ Firefox
- ✅ Edge
- ⚠️ WeChat browser (works, but data is isolated)
